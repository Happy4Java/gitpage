21年小结

发表于 2022-03-20 更新于 2023-03-04 分类于 发牢骚 Valine：

一、开篇

Hello大家好，我是哈皮。时间真的飞一般的流逝，转眼21年又过去了（22年也过去3个月了）。年后一直比较忙（不忙的时候也在偷懒），所以一直没有总结下21年的一些事情。一直觉得写20年总结的时候还没过去多久，但是想想已经过去一年多了（20年小结）。

二、回顾过去

去年订下的一些目标：

  • O1 健康与减肥

    • KR1 摘掉游泳圈
    • KR2 体重控制在63KG以下（包含），目前68.2。
    • KR3 早睡早起，不（少）熬夜。姑且把12点后睡当做熬夜吧。
    • KR4 每周一次健身房运动；每天坚持一些瑜伽垫运动。

  • O2 提升技术水平

    • KR1 脱离语言层面，更加深入的学习。
    • KR2 深挖Golang，希望Java少卷我一点，多一点Happy。
    • KR3 继续朗读并背诵Java八股文。
    • KR4 学习大数据全家桶。
    • KR5 坚持每周都写一篇博客。202104：写博客真的太费力，都是印象笔记里零散的点
    • KR6 坚持每天学习一点点。

  • O3 提高生活质量，让生活充满仪式感

    • KR1 多做饭，少点外卖，每天吃早餐。
    • KR2 换租一个房子，脱离公寓喧嚣的环境，永不再租公寓房。
    • KR3 带女朋友出去旅游一次，弥补20年的遗憾。
    • KR4 如果今年顺利，年末购置一辆梦想中的汽车。
    • KR5 在合适的时机，向女朋友求婚。

  • O4 工作

    • 工作上的想法和计划就不在这里多说了，希望今年工作顺利。

  • O5 投资理财

    • KR1 坚持自己制定的投资逻辑和计划，不动摇。
    • KR2 稳中求胜。
    • KR3 完成20年未完成的程序化投资。

看到这个列表，我都不好意思去打钩，丢人啊~

先说说健康与减肥吧，游泳圈是没有摘掉了，胖起来之后减肥真的很难，曾经就没想过自己会变成一个大肚便便的人，但是不知不觉中肚子真的就变大了。努力减几公斤，但是反弹却是那么的容易。早睡早起比之前改善了很多，熬夜也变少了。运动呢变成了饭后散步，健身房离开BIGO后就没去过了。

再说说技术篇，去年技术水平也是没什么提升的，一直想学的大数据呢还没开始，不过Golang、k8s、Python这些也算少有收获吧；博客全年几乎没怎么写，感觉自己也写不出东西，写博客最多还是大学和刚毕业那段时间，那时候什么都是新鲜的，一股脑的写，后来删掉了很多，后来也一直没有什么写博客的动力。

关于生活，去年完成了几件人生大事。比如：

  • 六一儿童节的时候提了心心念念的奔驰（其实之前老婆是想买宝马的，但是看了奔驰内饰就又想买奔驰，买车的前一天还在犹豫买奔驰还是买宝马）

  • 22年1月跟女票领了证，正式升级为一名丈夫

  • 比较遗憾的是没有出去旅游，其实机票都已经订好了，由于疫情又取消掉了。疫情已经三年，我们在一起一年就爆发了疫情，以至于我们都还没怎么出过远门旅游。

关于工作，今年4月在BIGO拿完年终奖就提了离职，BIGO这家公司一言难尽也不想去评价。离职之后就在“一家熟悉的公司”里一直远程办公了。

关于远程办公，只能说有好的地方，也有不好的地方。好的地方就是时间非常自由，结果导向，把事情做好就行。不好的地方就是跟同事的沟通效率比较差，很多时候没有面对面来得直接；其次就是不知道公司的一些状况。想起刚开始远程办公的时候，真的非常焦虑，特别是社保和公积金隔了两个月后才给我缴。不过随着越来越熟悉远程办公这种模式，现在已经能够游刃有余的处理各种问题了，也能够远程带人一起研发项目。

关于投资，今年买车后存款又掏空了，所以从年中开始至今就一直没有进行任何投资活动。投资这个事我会一直做下去，后面慢慢来吧。对于量化程序，其实写了个半成品，但是感觉这东西是没什么用的，投资耐心点就好。

2、关于游戏。今年是时隔四年重新燃起了LOL热情的一年，跟之前的老朋友一直双排开黑冲分，慢慢的找回了对游戏的乐趣。最初定位赛定位青铜三，然后在白银挣扎了一段时间，找回状态后打上了黄金，最后很挣扎的上了白金三，不过赛季末又掉回了白金四。不得不说真的老了，想当年大学的时候真的觉得自己玩得还行。也可能那时候无忧无虑就一心想着玩游戏吧，现在心思太复杂了。

大学的时候，蛇女、船长、卡牌、瑞兹、小法、吸血鬼等都是拿手英雄，不过感觉现在的版本很不适合自己，都是下水道英雄。

3、关于开车。买车的时候觉得自己一年开不了五千公里，结果半年就开了一万多公里，真的是买了车之后用车的需求就多了，带老婆回湖南的次数也多了，半年加了6k+的油费。

三、未来

22年，还是有颗拒绝当咸鱼的心，但是截止至今（2022年3月29日22:55:15）还是没有付诸行动，就连这篇寥寥草草的年终总结都拖拖拉拉了几个月还没写完。但是目标还是有的，谁知道哪天突然咸鱼翻身了呢是吧？
